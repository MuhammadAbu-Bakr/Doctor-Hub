# Doctor Hub

Doctor Hub is a Next.js healthcare portal foundation with JWT auth, role-aware protected routes, Prisma models for appointments and medical workflows, and a Render deployment blueprint.

## Local Setup

1. Install dependencies:

```bash
npm install
```

2. Copy `.env.example` to `.env` and update `DATABASE_URL` and `JWT_SECRET`.

3. Push the Prisma schema to your database:

```bash
npm run db:push
```

4. Start the app:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Useful Commands

```bash
npm run lint
npm run typecheck
npm run build
npm run render-build
npm run db:seed
```

## Render Deployment

This repo includes `render.yaml` for a web service and PostgreSQL database.

1. Commit and push the repository to GitHub.
2. In Render, create a new Blueprint from the repository.
3. Confirm the generated environment variables:
   - `DATABASE_URL` is wired from the Render PostgreSQL database.
   - `JWT_SECRET` is generated by Render.
   - `RESEND_API_KEY` can be left blank until email delivery is configured.
4. After the database is available, run the Prisma schema push once from Render Shell or a trusted local machine:

```bash
npm run db:push
```

The Render build command is `npm ci && npm run render-build`, and the start command is `npm run start`.
